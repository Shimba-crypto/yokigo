## 🛠️ THE VISION
**Yokigo** is a centralized Python-based environment designed for security simulation, game management, and terminal-based pranks. It is built with a "Cyber" aesthetic, prioritizing speed, stealth, and professional modularity.

---

## 🚀 CORE MODULES
The system is divided into four main protocols:

| Module | File | Description |
| :--- | :--- | :--- |
| **The Kernel** | `main_gui.py` | The Tkinter-based "Face" of the project. Manages navigation and auth. |
| **Security Gate** | `Yokigo-passes.py` | Handles SHA-224 encrypted token verification and user clearance. |
| **Fake Hack** | `hacked.py` | A high-speed visual simulator for matrix-style terminal effects. |
| **Boss Key** | `getaway.py` | The emergency purge protocol. Instantly hides work and opens decoys. |
